## Design Highlights

- **ExploredTileCache:** Client-side memory of terrain data for tiles no longer in StateView (665 KB max for full 128×128 map)
- **FogManager:** Centralized coordinator for tile visibility state, fog overlays, and camera bounds computation
- **Per-tile fog overlays:** Black (unexplored) / dimmed (explored) / normal (visible) rendering
- **Camera bounds:** Dynamically clamped to explored bounding box. Prevents scrolling into black void
- **Graceful Colyseus sync:** Reactive to onAdd/onRemove tile events with zero new server API requirements
