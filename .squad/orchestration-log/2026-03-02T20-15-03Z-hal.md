Rethink core gameplay loop — audit existing systems and propose redesign options to address the core problem: shape placement is not interesting enough.

## Work Done

1. **System Audit**
   - Documented current loop: place shape → get resources → place more shapes
   - Identified missing elements: tension, meaningful decisions, spatial puzzle, win condition
   - Confirmed creature interaction is decorative, not meaningful

2. **Three Redesign Proposals**
   - **Proposal A: "Habitat Puzzle"** (Dorfromantik-style biome-matching optimization)
     - Biome match scoring (2× income bonus for 3+ matching cells)
     - Cluster multiplier (4+ tiles = 1.5×, scaling to 3×)
     - Optional shape queue (randomized 3-shape queue, can skip for resource cost)
     - Creature attraction to clusters
     - Round structure (10-minute timer, score = territory count × efficiency)
     - **Scope: ~150 lines, no new schemas, smallest implementation cost**
   
   - **Proposal B: "Hungry Territory"** (Factorio/RimWorld colony pressure)
     - Territory upkeep cost (1 Wood per 5 tiles, or lose tiles)
     - Resource depletion (finite nodes, must expand)
     - Shape efficiency matters (each shape has tactical identity)
     - Creature threat (damage to tiles, defense geometry)
     - Rich resource nodes (visible objectives)
     - **Scope: ~120 lines, small-medium complexity**
   
   - **Proposal C: "Living Grid"** (Ecosystem colony sim)
     - Creatures settle on player territory (generate income)
     - Carnivores as defense against wild threats
     - Shape placement as habitat design (walls direct creature flow)
     - Population cap per biome (forces expansion)
     - Breeding returns (resource cost, acceleration)
     - Scoring weighted toward creatures (settled herbivores × 3, carnivores × 5)
     - **Scope: ~150 lines, 1 new schema field, medium complexity**

3. **Recommendation**
   - **Propose A (Habitat Puzzle)** as starting point
   - Rationale: smallest scope, fixes root cause (placement must be meaningful), composable with B/C, zero deletions, proven pattern
   - Implementation estimate: ~7 hours (or ~10h with shape queue + creature attraction)
