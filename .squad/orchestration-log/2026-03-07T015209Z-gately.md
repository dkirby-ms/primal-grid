- Falls back to full map bounds when no explored bounds set
- Integration: `main.ts` ticker pushes bounds from cache when dirty

## Key Design Decisions

1. **Cache-on-onAdd, not onRemove** — Prevents data loss if tile removed before we can capture structureType
2. **Pre-allocated Graphics per tile** — Avoids per-frame allocation churn; scales to 128×128 without perf issue
3. **structureType cached from day one** — Enables future silhouette rendering for explored structures
4. **Minimum camera bounds padding** — 10-tile extent prevents degenerate UX when exploring from 5×5 HQ start
5. **Text emoji icons for structures** — Consistent with existing HQ marker pattern; familiar UX
